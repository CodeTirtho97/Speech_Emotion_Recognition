This application uses advanced machine learning to detect emotions from speech audio files.

How it works:

  1. Upload an audio file (WAV, MP3, OGG, or FLAC)
  2. Our SVM model analyzes the audio features (MFCCs)
  3. Get instant emotion detection with confidence scores
